# Jeffrey Alan Young — Personal Site

A responsive static portfolio site for AI projects, publications, academic materials, and professional links.

## Run locally

Open `index.html` directly, or serve the folder with a local web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy

The folder can be deployed directly to GitHub Pages, Netlify, Cloudflare Pages, Replit, or any static web host.

## Files

- `index.html` — site content
- `styles.css` — responsive visual design
- `script.js` — mobile navigation, project filters, and reveal effects
- `assets/headshot.jpeg` — profile image
- `documents/` — downloadable resume, CV, research statement, and teaching statement
